# 🎮 RPG Shinzo Maker

Собственный движок для создания тактических RPG в духе Shining Force.  
Состоит из **игрового движка на Ruby** и **редактора карт на C + SDL2**.

---

## 🧩 Компоненты

### 1. Игровой движок (Ruby + Raylib)

| Требование        | Значение                     |
|-------------------|------------------------------|
| Язык              | Ruby 3.0+ / 4.0+             |
| Графика / Ввод    | Raylib 5.5+ (через гем `raylib-bindings`) |
| Платформа         | Windows / Linux / macOS      |
| Аудио             | Raylib Audio (MP3)           |
| Данные карт       | JSON                         |

**Возможности:**  
- Ходьба по карте с анимациями  
- Двухслойные карты (земля + декорации)  
- Система меню (Статус, Магия, Предметы, События)  
- Окно просмотра магии с иконками заклинаний

---

### 2. Редактор карт (C + SDL2)

| Требование        | Значение                     |
|-------------------|------------------------------|
| Язык              | C (GCC / MinGW)              |
| Графика           | SDL2 + SDL_image + SDL_ttf   |
| GUI               | Собственный (панели, диалоги)|
| Формат карт       | JSON (два слоя)              |
| Платформа         | Windows                      |

**Возможности:**  
- Два слоя тайлов (нижний — ландшафт, верхний — объекты)  
- Поворот / отражение тайлов  
- Назначение типов тайлов (проходимость)  
- Панорамирование и зум карты  
- Управление списком карт (создание, удаление, переименование, изменение размера)  
- Выбор фоновой музыки для карты  
- Сохранение путей к ресурсам в относительном виде  

---

## 🖱️ Управление зумом в редакторе

1. Наведи курсор на **область карты** (не на левую панель с тайлами).  
2. Зажми клавишу **Ctrl** (левый или правый).  
3. Крути колёсико мыши:  
   - **От себя** — приближение (зум увеличивается до 2.0×)  
   - **На себя** — отдаление (зум уменьшается до 0.5×)

Текущий масштаб отображается плавно, без рывков.
