редактор карт
Наведи курсор на область карты (не на левую панель с тайлами).

Зажми клавишу Ctrl (левый или правый).

Покрути колесо мыши:

От себя — приближение (зум увеличивается до 2.0×)

На себя — отдаление (зум уменьшается до 0.5×)
